WeatherTop Inc. produce a modular weather station called the WeatherTop 1000.

This low-cost device takes a set of meteorological readings and displays these on a simple LCD.

This is a Web companion application for the device, to displays collected readings separated by stations. Allowing user to view the readings, delete them or add new ones.
